# LifePlanner AI-RPG 🚀
**Gamified Productivity System with AI-driven Progression**

## 📌 Project Overview
LifePlanner is an Android application that transforms daily routines into an RPG experience. It uses Large Language Models (LLMs) to analyze user tasks and automatically translate real-world productivity into in-game character growth.

### 🤖 AI Integration (Core Feature)
The heart of the project is the **AI Logic Engine**. Instead of manual tagging, the app uses **Prompt Engineering** to:
- **Categorize Tasks:** Automatically detects if a task is physical (Strength), mental (Intellect), or routine (Agility).
- **Dynamic Balancing:** Calculates EXP rewards based on task complexity and user "streaks".
- **AI Dungeon Master:** (In development) Generates turn-based combat scenarios based on the user's weekly performance.

## 🛠 Tech Stack
- **Languages:** Java (Android SDK), C# (Unity - prototype)
- **Architecture:** MVC / Clean Architecture (principles)
- **AI Tools:** OpenAI API / Gemini API (via Prompt Engineering)
- **Database:** SQLite (Room persistence - in progress)
